# Solar Plant Performance & Fault Analysis Dashboard

This project analyzes solar plant performance using Python, SQL, and Power BI to identify inefficiencies, fault patterns, and environmental impacts on energy generation.

## Objective

- Analyze solar power generation data
- Identify performance drops and inefficiencies
- Detect fault patterns and their impact
- Understand environmental effects (irradiation, temperature)

## Tech Stack

- [Python](https://github.com/BikalpaRegmi/Energy_Data_Project/tree/main/Python)
- [SQL](https://github.com/BikalpaRegmi/Energy_Data_Project/tree/main/SQL)
- [Power BI](https://github.com/BikalpaRegmi/Energy_Data_Project/tree/main/DashBoards)

## Project Workflow

1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. Feature Engineering (Power Loss, Time Categories)
4. PDF SCRAPING
5. SQL-based Data Analysis
6. Dashboard Creation in Power BI

## Key Insights 

- There are total of 68 fault panels and 2 of them probably are on verge of being
damaged.
- The conversion of DC to AC power is strongly efficient.
- The faults occurs mainly on peak afternoon (12pm-3pm).
- The performance ratio stayed average around 20-25% throughout month.
- The performance peaked at peak afternoon at 11pm-1pm.
- The module temperature and performance were highly correlated.
- The overall DC_Power and Performance performed well in high irridation
despite some panel being fault.
